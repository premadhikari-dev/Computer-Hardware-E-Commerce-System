CHES is a full-stack e-commerce web application for selling computer hardware products.
It includes user authentication, cart system, order processing, admin dashboard, and eSewa payment integration.

## ✨ Features

- User registration & login
- Product browsing by category
- Search products integration
- Shopping cart system
- Checkout system
- eSewa payment integration
- Order management
- Admin product management
- Soft delete & restore system
- Stock validation system

## 🛠 Technologies Used

- PHP (Core PHP)
- MySQL
- HTML5
- CSS3
- JavaScript
- PDO (Secure Database Access)
- eSewa Payment API

## ⚙️ Installation Guide

1. Clone the repository
   git clone [https://github.com/Eto0001/Computer-Hardware-E-Commerce-System]

2. Move project to htdocs (XAMPP)

3. Import database from /database/ches.sql

4. Configure database in src/db.php

5. Run on:
   http://localhost/CHES_PROJECT/public
